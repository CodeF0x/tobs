Project Goal

TOBS shows you the most important information about your system in an instant using comprehensible charts.
With customizable options like refreshrate, dark- and light mode, drag- and drop alignment of the charts and some more, you can make Tobs your very own experience.

What TOBS can do

Currently, Tobs is able to monitor CPU, RAM, Disk, Network and GPU* usage.

It nicely displays all the data received from the components in a great looking interface.
